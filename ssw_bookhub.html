SSW BookHub

A Mobile E-Book Store Application

React Native · Supabase · Unity 3D · Python (LLM/TTS)

View App Overview & User Guide


Key Features

👤 Customer Features
  • Registration & Login (with preference selection)
  • Browse by categories & search
  • Shopping cart & order placement
  • Upload payment proof (PayMe/WeChat)
  • View order history & track status
  • Rate & comment on purchased books
  • View browsing footprint & manage preferences
🛠️ Admin Features
  • Add / edit products (with image upload)
  • Manage inventory (restock, edit details)
  • Process orders (approve payment, ship, update location)
  • Sales analytics (charts & top‑rated products)
  • View customer details & purchase history
  • Real‑time order chat with customers
🤖 3D Chatbot
  • Unity 3D character with lip‑sync
  • Voice interaction (TTS + STT)
  • Personalized recommendations based on preferences
  • Integrates with LLM (CantoneseLLM) for Q&A

Technical Architecture

React Native
Supabase
Unity 3D
Python
  • Frontend React Native – cross‑platform mobile app (iOS/Android/Web) with a responsive Chinese UI.
  • Backend Supabase – PostgreSQL database, authentication, file storage, and real‑time subscriptions for orders and chat.
  • 3D Chatbot Unity 3D – animated avatar with lip‑sync and voice interaction (TTS/STT) for an engaging user experience.
  • AI Services Python backend – integrates CantoneseLLM for natural language Q&A and custom TTS for voice output.
  • Real‑time Communication WebSocket (via Supabase) – instant order status updates and messaging between customers and admins.
