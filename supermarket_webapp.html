Key Features

🔐 Authentication
  • Google OAuth 2.0 & Local Strategy (Passport.js)
  • Role-based access (USER / STAFF)
  • Secure session management with MongoDB store
🛒 Product & Cart Management
  • Full CRUD for products (create, read, update, delete)
  • Search, filter by category, and pagination
  • Shopping cart with quantity adjustments
  • Checkout & invoice generation (PDF-ready)
📡 RESTful APIs
  • Product APIs (GET, POST, PUT, DELETE)
  • Cart APIs for guest and authenticated users
  • Search & filter endpoints
  • Fully documented with curl examples


Technical Architecture

Node.js
Express
MongoDB Atlas
Passport.js
EJS
Render
  • Backend Node.js + Express – RESTful API server with modular controllers and middleware for authentication and validation.
  • Database MongoDB Atlas – NoSQL database with collections for users, products, carts, and permissions.
  • Authentication Passport.js – Google OAuth 2.0 and local strategy, with session persistence using MongoDB store.
  • Frontend EJS templates – server‑side rendering with responsive navbar, product listing, cart, and invoice pages.
  • Deployment Render – cloud hosting with environment variables for secure configuration.
  • Version Control Git – collaborative development with group contributions.
