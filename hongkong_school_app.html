Project Overview

Hong Kong School Info is dedicated to providing parents and students with a one-stop Hong Kong school search experience. It integrates comprehensive kindergarten, primary, and secondary school data from the official Education Bureau (EDB) dataset, making educational resource comparison and selection simpler and more intuitive than ever.

Key Features

  • Smart Search – Find schools by name or district with real-time bilingual filtering
  • Favorites – Save preferred schools to your list with offline storage (AsyncStorage)
  • Comparison – Multi-dimensional side-by-side school comparison
  • Map View – Interactive school locations with markers and tooltips
  • Dark Mode – Full system-level dark theme support
  • Bilingual – Seamless Chinese & English switching

Methods

  • Data Collection – Official EDB JSON dataset used directly; structured and filtered dynamically within the app
  • Cross-Platform Architecture – Built with React Native using React Hooks, AsyncStorage, and React Native Maps
  • UI/UX Design – Modern minimalism with dark/light mode, bilingual support, and accessibility-tested contrast

Technology Stack

Framework: React Native (cross-platform iOS/Android)
UI Libraries: React Native Paper, React Native Maps
Storage: AsyncStorage (offline favorites & language preferences)
State Management: React Hooks (useState, useEffect, useRef)
Device APIs: Linking (open school websites), MapView
Data Source: Official EDB JSON dataset
